# Clinic Dashboard

Open `index.html` in any web browser — just double-click it, no install and no internet connection needed — to see every consultation from all of your Filler Mirror iPads in one place. On each iPad, go to the Staff panel → Sessions → Export JSON to save that device's consultation file, then drag every file you've collected onto this page (or click the drop area to pick them from a folder): the dashboard merges them, automatically skips anything you drop twice, and shows you totals, trends, and a searchable, sortable table across every device combined. If you've also used the separate dataset tool to calibrate the engine, you can drop its "Download engine params" JSON into the second panel near the bottom to see what those numbers mean in plain language. Everything happens locally in your browser — nothing is ever uploaded anywhere — which also means it's temporary: the page forgets everything when you close the tab, so keep your exported JSON files and just re-drop them next time you want to look at the numbers (or use the "Export merged sessions JSON" button to save a combined file first).
